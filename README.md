![header](https://capsule-render.vercel.app/api?type=Venom&color=auto&height=300&section=header&text=IoTerminal%20&fontSize=90)



## 👀 서비스 소개
* 서비스명:  IoTerminal
* 서비스설명: IoT 빅데이터 수집 및 관리 시스템
<br>

## 📅 프로젝트 기간
2025.04.22 ~ 2025.06.04 (6주)
<br>

## ⭐ 주요 기능
* IoT센서의 스트림
* 워크플로우 연결을 통한 데이터 관리 
* IoT디바이스 관리
